# 中文小说项目
 > 如：tc、ft、ky等
1. 迁移广告sdk时，applovin 替换
   1. 替换build.gradle中的apiKey（ad-review-key）
   2. 替换sdk key
      2.1 Androidmanifest中（如果有）
      2.2 AdFactory中（如果有）
   3. 删除rootProject中的 .safedk目录！！！
   4. 执行clean project！！！
   5. 执行make project
2. pangleId 在Application中替换
3. 中文项目的key基本都在keystore.properties或app的gradle中！
4. 如项目有anr，可引入线程池或者kotlin协程进行优化
   1. google控制台中可查看具体anr
   2. 主线程耗时操作
   3. io操作
   4. 主线程频繁操作数据库
5. 升级数据库：建议使用ObjectBox，同greenDAO一团队开发，迁移成本较低；

# AutoApp
- 项目托管在gitee
- 核心任务类：TaskPollingManager
- 任务中基本分两种类型：
  - 按步骤顺序执行+步骤回滚，如：TC、LM等任务
  - 状态机驱动，如google登录
